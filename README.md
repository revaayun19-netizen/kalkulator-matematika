# kalkulator-matematika# 🧠 MathMaster Calculator

## Deskripsi

MathMaster Calculator adalah aplikasi kalkulator matematika interaktif berbasis Python dan Streamlit yang dirancang untuk membantu pengguna menyelesaikan berbagai perhitungan matematika dalam satu aplikasi. Selain operasi aritmatika dasar, aplikasi ini juga menyediakan fitur faktorial, permutasi, kombinasi, FPB, KPK, pangkat, dan akar.

Aplikasi ini cocok digunakan sebagai media pembelajaran matematika bagi siswa maupun mahasiswa karena memiliki antarmuka yang sederhana, interaktif, dan mudah digunakan.

## Fitur Utama

### ➕ Operasi Dasar

* Penjumlahan
* Pengurangan
* Perkalian
* Pembagian

### 🔢 Faktorial

* Menghitung nilai n!

### 📊 Permutasi dan Kombinasi

* Menghitung nPr
* Menghitung nCr

### 📈 Pangkat dan Akar

* Perhitungan pangkat
* Perhitungan akar kuadrat

### 📐 FPB dan KPK

* Menentukan Faktor Persekutuan Terbesar (FPB)
* Menentukan Kelipatan Persekutuan Terkecil (KPK)

## Manfaat Aplikasi

* Membantu pembelajaran matematika secara interaktif.
* Mempermudah perhitungan matematika sehari-hari.
* Menjadi media latihan konsep kombinatorika dan teori bilangan.
* Meningkatkan pemahaman mahasiswa terhadap implementasi matematika menggunakan Python.

## Pengembang

Proyek ini dibuat sebagai aplikasi pembelajaran matematika berbasis Python dan Streamlit dengan konsep kalkulator multifungsi yang sederhana dan mudah digunakan.
